# 🌿 ETF 定期定額複利試算工具

這是一個簡潔、互動式的 ETF 試算網站，幫助你模擬長期投資的資產成長，並提供即時股票查詢功能。  
由 [訊丞科技]製作，採用自然綠系風格，讓理財變得更直覺、更安心。

---

## 🚀 功能特色

- 📈 **ETF 複利試算**：輸入每月投入金額、年化報酬率與年數，即時顯示資產成長與總投入折線圖
- 🧠 **互動式圖表**：滑鼠移動顯示 tooltip，清楚標示每年資產與投入金額
- 🔍 **股票資訊查詢**：輸入股票代碼，一鍵開啟 Yahoo 股市頁面
- ⭐ **熱門股票快捷鍵**：快速查詢 0050、2330、00878 等熱門標的
- 🌗 **深色模式切換**：支援深色 / 淺色主題，保護眼睛
- 📥 **圖表下載功能**：一鍵下載圖表為 PNG 圖片，方便分享與記錄

---

## 🛠 使用方式

1. 開啟網站 👉 [https://zkin0128.github.io/etf-calculator](https://zkin0128.github.io/etf-calculator)
2. 輸入每月投入金額、年化報酬率與投資年數
3. 查看圖表與試算結果
4. 使用股票查詢功能快速查看個股資訊
5. 點擊右上角「🌓」切換深色模式
